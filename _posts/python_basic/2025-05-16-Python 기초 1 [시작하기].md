## 통합 개발 환경 설치
IDE(통합 개발 환경, Integrated Development Environment)은 코드 작성, 실행, 디버깅, 프로젝트 관리 등 개발에 필요한 기능을 한 곳에서 제공하는 프로그램이다.

<br>

### 1. Visual Studio Code

VSCode 설치: https://code.visualstudio.com/Download
- 설치 후 좌측 사이드바의 확장(Extension) 아이콘 클릭 → 'python' 검색 → 'Python', 'Pylance' 등 필수 확장 설치.

**특징**
- 마이크로소프트에서 개발한 무료 오픈소스 코드 에디터
- 파이썬뿐 아니라 다양한 언어 지원
- 가볍고 빠르며, 확장 프로그램(Extension)으로 기능을 자유롭게 추가 가능
- 통합 터미널, Git 연동, 풍부한 테마와 플러그인 제공
- 초보자부터 전문가까지 모두 사용하기 적합

<br>

### 2. PyCharm

PyCharm 설치: https://www.jetbrains.com/pycharm/download/

**특징**
- JetBrains에서 개발한 파이썬 전용 IDE
- 강력한 코드 자동 완성, 디버깅, 테스트, 코드 품질 검사, 프로젝트 관리 기능 제공
- 데이터 분석, 웹 개발 등 다양한 파이썬 프로젝트에 최적화
- Git, 데이터베이스 등 다양한 개발 도구와 통합
- 무료(Community)와 유료(Professional) 버전 제공

<br>

### 3. Spyder

아나콘다 설치 시 함께 설치 된다.

**특징**
- 과학 계산, 데이터 분석에 특화된 오픈소스 파이썬 IDE
- MATLAB과 유사한 인터페이스로 과학자, 데이터 분석가에게 인기
- 코드 편집기, 변수 탐색기, 인터랙티브 콘솔, 강력한 디버깅 도구 제공
- Numpy, SciPy, Matplotlib 등 데이터 분석 라이브러리와 연동이 편리

<br>

# 파이썬 기본 개념

---

**문장**  
실행 가능한 코드 최소 단위  

**프로그램**  
문장들이 모여 프로그램을 구성  

**표현식**  
어떠한 값을 만들어 내는 간단한 코드

<br>

## 연산자(Operators)
데이터의 연산을 수행하는 기호이다.  
연산자의 종류는 다음과 같다.  
- 산술 연산자 (Arithmetic Operators)
- 대입 연산자 (Assignment Operators)
- 비교 연산자 (Comparison Operators)
- 논리 연산자 (Logical Operators)
- 비트 연산자 (Bitwise Operators)
- 멤버 연산자 (Membership Operators)
- 식별 연산자 (Identity Operators)

<br>

## 키워드(Keyword)
<u>예약어</u>라고도 하며, 파이썬에서 고정적인 기능을 담당하도록 예약된 단어이다.

```python
# 키워드 예시
break
int
self
for
```

<br>

## 식별자(Identifier)
프로그램 내에서 어떤 대상을 유일하게 구별할 수 있도록 지정한 이름을 의미한다.

- 식별자 생성 규칙:  
    1) 키워드 사용금지  
    2) 특수문자는 언더바(_)만 허용  
    3) 숫자로 시작 불가  
    4) 공백 포함 불가  
    5) 한글은 권장 하지 않음  
    6) 대소문자 구분  

좋은 식별자 이름을 짓는 것은 코드의 가독성을 높이기 때문에 명확하게 짓는 것이 좋다.  

**식별자 작성 스타일**  
식별자가 2단어 이상으로 되어 있을 때 가독성을 높이기 위해 사용되는 작성법이다.
- Snake Case: 단어 사이에 언더바로 구분해서 작성

```
two_stone_plates
```

- Camel Case: 각 단어의 첫 글자만 대문자로 작성

```
twoStonePlates
```

반드시 지켜야 하는 것은 아니지만,  
프로그래밍 언어마다 개발자들 사이의 암묵적인 명명 규칙이 존재한다.  

파이썬에서는 snake case를 사용한다.

<br>

## 주석
주석처리한 문장은 파이썬 인터프리터가 무시한다.

- `#`을 이용한 주석처리

```python
# 이 문장은 파이썬 인터프리터가 무시합니다.
```

- `'`, `"` 3개 쌍으로 감싸서 주석처리  

```python
'''
여러 줄을
동시에
주석처리 가능
'''

"""
여러 줄을
동시에
주석처리 가능
"""
```

`"""` 쌍은 나중에 배울 함수와 클래스의 설명을 작성할 때도 사용된다.  

<br>

## 자료  
리터럴(literal)이라고도 표현하며, 값 자체를 의미한다.  

```python
3.14
100
'a' # 파이썬에서 문자는 따옴표로 표현.
"가나다" # 작은 따옴표, 큰 따옴표 모두 사용 가능. 
```

<br>

## print()  
print()는 <u>함수</u>이며, 파이썬에서 가장 기본적인 출력 방법이다.

```python
print("Hello World")
```
  
쉼표로 구분된 요소는 띄어쓰기로 출력한다.  
여러 값을 한번에 출력할 때 사용된다.
```python
print(10, 20, 30, 40, 50)
```

**함수**  
하나의 단위 기능 묶음

- 함수는 `식별자` + `()` 의 모양으로 되어있다
- 함수의 `()` 안에 함수 실행의 대상물(arguments)이 들어가게 된다.

빈 괄호인 print()는 줄바꿈의 효과가 있다.

```python
print("# 아무것도 출력하지 않습니다.")
print("---확인 출력선---")
print()     # 줄바꿈으로 기능
print()
print("---확인 출력선---")
```

**종료코드**  
실행 후 결과 창에 `[ 종료 코드 0 ]`는 문제 없이 실행 됐음을 알려주는 것이다.  
문제가 발생했다면 에러 정보와 `0`이 아닌 다른 숫자가 나온다.
