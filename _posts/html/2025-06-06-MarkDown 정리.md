# 이미지 첨부

```html
![이미지 설명](경로)
```

## 이미지 크기 조정

```html
<img src="./image.png", height="100x", width="100px">
```

## 이미지 정렬
**왼쪽/오른쪽 정렬**

```html
<img align="left" src="./image.png">
<img align="right" src="./image.png">
```

**가운데 정렬**

```html
<p align="center">
 <img src = "./image.png">
</p>
```
